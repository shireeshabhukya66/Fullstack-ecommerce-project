# Full Stack Ecommerce Web Application

A full-stack ecommerce web application developed using React.js and Django REST Framework with JWT Authentication.

## Features

- User Registration and Login Authentication
- JWT Token Authentication
- Protected APIs using Django REST Framework
- Product CRUD Operations
- Search Functionality
- Category Filter
- Pagination
- Responsive UI using Bootstrap
- Dark Mode Toggle
- React Router Navigation

## Technologies Used

### Frontend
- React.js
- Bootstrap
- JavaScript
- HTML5
- CSS3

### Backend
- Django
- Django REST Framework
- JWT Authentication
- MySQL
